# マレットゴルフ スコアアプリ Ver.2.0 試作初版（完成版）

GitHub Pages用の6ファイル一式です。

## ファイル
- index.html
- manifest.webmanifest
- sw.js
- icon-192.png
- icon-512.png
- README.md

## 主な仕様
- 4〜7人対応
- 5〜7人は横画面推奨
- 1〜8と「＋」でスコア入力
- 入力後、自動で次の人へ移動
- バーディは赤、イーグル以下は青
- 入力済みセルはクリーム色
- 各ホールに「全員同じ」
- メンバー名の差し替え
- 4人分の名前枠を常時表示
- 合計欄を大きく表示
- A4横向き印刷対応
- 端末内への保存・読込
- iPhone・Androidのホーム画面追加対応

## GitHubへの更新方法
6ファイルをすべて選択し、GitHubの mallet-golf リポジトリへアップロードして上書きしてください。
